# ふくフォト

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

ふくフォト is a web application that allows users to upload, view, and share photos of local specialty products. Built with React, TypeScript, and Vite.

## Features

-   Upload and share photos of local specialty products
-   View recommended specialty product photos
-   Discover new specialty products with the "gacha" feature
-   View a ranking of popular specialty product photos

## Tech Stack

-   **Frontend:** React, TypeScript
-   **UI Library:** Chakra UI
-   **Routing:** React Router
-   **Build Tool:** Vite

## Getting Started

### Prerequisites

-   Node.js
-   npm

### Installation & Running Locally

1.  Clone the repository:
    ```bash
    git clone https://github.com/202207-hackathon-team3/front.git
    ```

2.  Navigate to the project directory and install dependencies:
    ```bash
    cd front
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

The application will be available at `http://localhost:3000`.
