# ふくフォト

ふくフォトは、ユーザーが地元の特産品の写真をアップロード・閲覧・共有できるWebアプリケーションです。React、TypeScript、Viteを使用して構築されています。

## 機能

- 地元の特産品の写真をアップロードして共有
- おすすめの特産品写真を閲覧
- 「ガチャ」機能で新しい特産品を発見
- 人気の特産品写真ランキングを閲覧

## 技術スタック

- **フロントエンド:** React, TypeScript
- **UIライブラリ:** Chakra UI
- **ルーティング:** React Router
- **ビルドツール:** Vite

## はじめに

### 前提条件

- Node.js
- npm

### インストールとローカルでの実行

1. リポジトリをクローンします:
    ```bash
    git clone https://github.com/202207-hackathon-team3/front.git
    ```

2. プロジェクトディレクトリに移動し、依存関係をインストールします:
    ```bash
    cd front
    npm install
    ```

3. 開発サーバーを起動します:
    ```bash
    npm run dev
    ```

アプリケーションは `http://localhost:3000` で確認できます。
